# Git101: Git & GitHub for Data Engineering Teams

Public training repository for Git101. Learners should use this as a template, create their own copy, complete labs chapter by chapter, and submit PR links or screenshots in Zoho Learn.

## Learner workflow

1. Click **Use this template**.
2. Create your own repository, for example `git101-firstname-lastname`.
3. Clone your own repository locally.
4. Complete labs under `labs/`.
5. Use feature branches.
6. Open pull requests in your own repo.
7. Submit evidence in Zoho Learn.

## Data and security policy

Use only synthetic data. Do not commit real client data, passwords, API keys, cloud credentials, `.env` files, raw production data, large generated files, virtual environments, logs, or output folders.
